## Steps to run the app:
1. Copy file ```db.json.template``` to ```db.json``` i.e. run: ```cp db.json.template db.json```
2. Run ```npm install```
3. Run ```npm run json:server```
4. In another terminal window, run ```npm start```. This will serve the app on http://localhost:3000 and open it in the web browser for you!
